My Latest Scratch Project


I was inspired to create this game by similar space shooting games such as galaga or space invaders. I have used some of scratch's forums for inspiration on how to do certain things. I created the "pew" noise heard upon firing the blaster using audacity as a wav file. Wav is uncompressed and did not require any other form of download, as opposed to the mp3 files which required their own application due to it's compressed nature. I used audacity to record and clip the sound to the size I wanted. I created the blaster and the rocket graphics using the paint new sprite option in scratch. These graphics were in the vector form, which differs from raster in that they are precise down to each pixel, where a raster graphic may differ as it is not represented by formulas. I did this by combining simple shapes into those more fitting for the game. I used and, or and not all to modify the flight path of the blaster. I used this to determine where the blaster would fly and under which conditions.Boolean logic simply states that something is either a 1 or a 0, on or off respectively. Number systems are used to describe numbers. These number systems put values into forms that a computer can interpret.
